Software Development Company Minimum Viable Product

  • Sep 21, 2014
  • Categories: Blog
  • Tags: #containers #soa #bullshit-bingo

What's the bare minimum you need for a software development house to run a service oriented architecture?

Requirements

Separate modules with defined dependencies

Each different bit of code is split up (it's own git repo).

It can be built independently (though it may depend on other modules, there are no cyclic dependencies).

Two kinds of modules:

  • applications
  • libraries

Applications depend upon libraries.

Containers built from modules

A container will contain a single application module
